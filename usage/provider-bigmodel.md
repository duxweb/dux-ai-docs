# BigModel 服务商

BigModel 在当前 Dux AI 里适合用于：

- 对话模型
- Embedding 模型
- 图片生成
- 视频生成

## 注册地址 / 控制台入口

- 智谱开放平台：`https://open.bigmodel.cn/`
- 官方文档：`https://docs.bigmodel.cn/`

![BigModel 平台首页或控制台占位图](/images/provider-bigmodel-console.png)

## 怎么获取 API Key

智谱官方文档已经将“注册账号、环境配置、获取 API Key”作为快速开始的一部分。推荐步骤：

1. 访问 `open.bigmodel.cn`
2. 注册或登录账号
3. 开通对应模型服务
4. 进入 API Key 管理页面
5. 创建并复制 API Key

![BigModel 获取 API Key 占位图](/images/provider-bigmodel-api-key.png)

## 适合什么情况

- 你希望使用智谱体系能力
- 你需要国内模型平台的统一接入方式
- 你希望同时覆盖聊天和多媒体能力

## 推荐接入顺序

1. 先接对话模型
2. 再接 Embedding
3. 再接图片与视频
4. 最后再放进工作流或复杂业务里

## 推荐做法

先把基础聊天跑通，再逐步加多媒体能力，不要第一次就同时接太多类型，否则排障成本会很高。
