# 机器人绑定与排障

平台参数配好之后，还差最后两步：

- 把机器人绑定到智能体
- 验证消息是否真的能走通

![机器人绑定智能体占位图](/images/bot-bind-agent.png)

## 绑定顺序建议

推荐按这个顺序验证：

1. 先确认智能体本身能在后台正常对话
2. 再创建并保存机器人
3. 配好平台回调地址
4. 把机器人绑定到智能体
5. 在平台里发送消息做联调

## 排障顺序

如果机器人收不到消息或不能回复，建议按顺序检查：

1. 回调地址是否正确
2. 平台密钥是否填写完整
3. 机器人是否启用
4. 是否已经绑定智能体
5. 智能体本身是否可正常对话
6. 工作流、能力或知识库是否导致中间执行失败

## 推荐测试方式

第一次联调时，不要直接测复杂业务。

建议先发一条最简单的消息，例如：

```text
你好，请介绍一下你自己
```

如果这一步都没通，先不要继续测试知识库、工作流、图片或视频能力。
