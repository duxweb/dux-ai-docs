# 智能体概览

智能体是 Dux AI 面向用户的核心执行单元。

你可以把它理解成：

- 有角色设定
- 有模型支撑
- 有能力扩展
- 能接机器人
- 能调用知识库和工作流

## 智能体能力怎么理解

智能体能力并不是固定死的，而是可以按场景逐步挂载。

最常见的能力类型包括：

- 知识库检索
- HTTP 请求
- 函数调用
- MCP 调用
- 图片生成
- 视频生成
- 文档解析
- 延迟消息
- 通知发送
