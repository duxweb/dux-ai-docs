# 图片生成能力


图片生成属于智能体可挂载的能力之一，也可以由工作流调用。

## 适合什么场景

- 海报初稿
- 商品图灵感图
- 内容配图
- 概念图生成

## 后台里通常要配什么

### 图片模型

这是必填项，决定用哪个图片模型生成。

### 默认生成数量

- 默认值：`1`
- 一般先保持默认

### 默认尺寸

- 默认值：`1920x1920`
- 可以按业务需要调整

### 默认负向提示词

可选项，用来告诉模型“不要出现什么内容”。

## 智能体输入通常支持什么

这个能力最核心的输入是：

- `prompt`：图片提示词（必填）

可选输入通常还包括：

- `image`：图生图输入
- `mask_url`：遮罩图
- `size`：尺寸
- `n`：生成数量
- `negative_prompt`：负向提示词

## 更适合放哪里

- 简单生成：可直接作为智能体能力
- 复杂流程：更适合放进工作流

## 推荐的最小配置

第一次使用时，建议先这样配：

- 选一个图片模型
- 默认生成数量保持 `1`
- 尺寸先用默认值
- 先不填负向提示词

这样最容易快速跑通一张图。
