# 函数调用能力


适合把系统内部业务动作封装成函数，让智能体按需调用。

## 适合什么场景

- 创建业务记录
- 触发某个模块动作
- 执行内部查询
- 做可控的业务操作

## 后台里通常要配什么

### 选择函数

这是必填项。你要先从已注册函数里选一个。

### 函数入参（JSON）

这里填写要传给函数的参数。

- 默认值：`{}`
- 建议先从最简单的 JSON 开始

## 使用建议

- 优先把函数定义清晰、参数收敛
- 不要一次挂太多复杂函数
- 先确认函数本身可以独立执行，再挂给智能体

## 推荐的最小配置

第一次使用时，建议先这样配：

- 先选一个最简单的函数
- 函数入参先用 `{}`
- 先确认函数本身能成功执行

先跑通一条最简单的调用，再继续接复杂业务动作。
