# 解析配置

解析配置决定文件能不能被正确读出来，尤其是 PDF、图片、混合文档这类场景。

## 这一页适合看什么

这一页主要告诉你：

- 为什么它对知识库和附件处理很重要

## 适合哪些文件

当前文档侧重介绍：

- PDF
- PNG / JPG / JPEG / WEBP / BMP / GIF

## 为什么解析配置很重要

因为知识库效果不只是看模型，还取决于资料有没有被正确提取成文本。

## 推荐做法

- 第一次只用少量文件测试
- 先确认解析内容是否完整
- 再进入知识库入库和检索阶段
